# CustomCleaner
プレイヤーのアイテムを綺麗さっぱり

## Features
アイテムを指定することで該当するアイテムを指定したプレイヤーから取り除くことが出来る

## Usage
ストレージの"senba.clean:Item"に通常どうりアイテムの状態をかいていけばいい。(Countはbyte型以外も対応)
そして、取り除きたいプレイヤーに"function senba.clean:main"を実行させる。
